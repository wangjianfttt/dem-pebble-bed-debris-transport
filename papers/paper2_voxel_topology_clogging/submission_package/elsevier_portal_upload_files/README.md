# Paper 2 Elsevier Portal Upload Files

This directory contains a minimal portal-facing upload package. It intentionally excludes internal reviewer-risk maps, fallback-journal materials, audit ledgers and author-confirmation requests.

## Contents

- `manuscript`: Main manuscript, source files, bibliography and highlights.
- `main_figures`: All figure PDFs referenced as main-text figures.
- `graphical_abstract`: Graphical abstract in raster, editable/vector and Word-preview formats.
- `supplementary_material`: Formal supplementary text, tables and supplementary-figure PDFs.
- `cover_letter`: Journal-facing cover letter in Word format plus markdown source.
- `portal_copy_text`: Copy-ready text helpers for the submission system; not intended as scientific evidence files.

## Boundary

- Use the separate internal `journal_submission_files` archive for reproducibility and audit handoff.
- Use this portal package for files that may be uploaded or copied into the Elsevier submission system.
