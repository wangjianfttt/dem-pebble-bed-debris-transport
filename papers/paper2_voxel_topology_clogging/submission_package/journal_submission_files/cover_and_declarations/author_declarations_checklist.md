# Paper 2 Author Declarations Checklist

This checklist is for author confirmation before journal submission. It intentionally separates author-approved facts from draft wording generated during manuscript preparation.

## Author Approval

| Item | Required confirmation |
|---|---|
| Final title | All authors approve the final title. |
| Author order | All authors approve the author order in the manuscript. |
| Affiliations | Each author confirms their affiliation is correct. |
| Corresponding author | Corresponding author name and email are confirmed. |
| ORCID | ORCID IDs are collected if required by the journal system. |

## Scientific Scope

| Item | Required confirmation |
|---|---|
| Numerical scope | Authors agree that the paper is framed as a DEM-based numerical study. |
| Voxel wording | Authors agree not to describe the voxel fields as experimental CT data. |
| Clogging criterion | Authors agree that the current `Ib` is pressure-free structural pre-screening. |
| Critical transition | Authors agree not to claim a universal critical clogging transition. |
| Gas coupling | Authors agree that one-way Stokes drag is stated and pore-resolved gas redistribution is not claimed. |

## Declarations

| Declaration | Draft status | Author action |
|---|---|---|
| Funding | Placeholder in manuscript | Confirm grant numbers or no-funding wording. |
| Competing interests | Draft says none known | Each author must confirm. |
| Acknowledgements | Partial draft | Confirm computing resources, collaborators and institutional wording. |
| Data availability | Repository DOI/accession not assigned | Deposit package and insert DOI/accession. |
| Code availability | Repository DOI/accession not assigned | Confirm code deposit contents and license. |
| AI/tool-use disclosure | Not finalized | Check journal and institutional policy. |

## Data And Code Deposit Gate

Before submission, confirm that the public deposit contains:

- manuscript source and compiled preview;
- processed source data and figure tables;
- figure-generation scripts;
- shared DEM post-processing, voxel, topology, transport and model utilities;
- supplementary tables and supplementary-figure catalogue;
- tests sufficient to reproduce the processed figures and readiness audits;
- optional raw DEM dump/restart archive or a clear statement explaining where large files are stored.

## Required Final Action

The manuscript should not be submitted until this checklist is reviewed by the corresponding author and the final DOI/accession, funding statement, competing-interest statement and acknowledgements are inserted into the manuscript.
